# SoundCloud Desktop

A beautiful native desktop client for SoundCloud with a custom glass theme.

![SoundCloud Desktop](https://img.shields.io/badge/SoundCloud-Desktop-orange?style=for-the-badge&logo=soundcloud)
![Tauri](https://img.shields.io/badge/Tauri-2.0-blue?style=for-the-badge&logo=tauri)
![Rust](https://img.shields.io/badge/Rust-Language-orange?style=for-the-badge&logo=rust)

## Features

- Full SoundCloud experience in a native app
- Custom glassmorphism dark theme
- Animated background
- Premium UI with blur effects
- Fast and lightweight (Tauri + Rust)
- Available for macOS and Windows

## Installation

### macOS
Download the `.dmg` file from [Releases](https://github.com/PRISSET/SoundClaudeDesktop/releases) and drag to Applications.

### Windows
Download the `.msi` installer from [Releases](https://github.com/PRISSET/SoundClaudeDesktop/releases) and run it.

## Development

### Prerequisites
- [Rust](https://rustup.rs/)
- [Node.js](https://nodejs.org/)

### Setup
```bash
git clone https://github.com/PRISSET/SoundClaudeDesktop.git
cd SoundClaudeDesktop
npm install
npm run tauri dev
```

### Build
```bash
npm run tauri build
```
